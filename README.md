# Product Name: InsightU
### Developed by Team CodeSpark (Subhajit Mandal)

An intelligence-sharing platform designed to democratize institutional knowledge, hidden recruitment pipelines, and the "invisible curriculum" for first-generation college students.

## Core Features
- **Structured Intel Submission:** Peer-reviewed crowd-sourcing for department norms, faculty preferences, and deadlines.
- **Credibility Scoring & Verification:** Algorithmic trust layer using peer verification to filter out noise.
- **Branch & Urgency Tagged Feeds:** Personalized, contextual dashboards tailored to academic calendars.
- **Proactive Nudge Engine:** Smart notifications alerting students before critical windows close.
